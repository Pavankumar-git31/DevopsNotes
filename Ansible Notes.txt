ANSIBLE:

ANSIBLE MODULES:
modules works on key value pair.
modules used to complete the work which we want 
depend on the work module will change 

how to use modules :
install the git in all worker nodes : ansible all -m yum -a "name =git ,state=present"
delete the git in all worker nodes : ansible all -m yum -a "name =git , state=absent"
install latest version of something : ansible all -m yum -a "name=git , state=latest"
copy the file to all worker nodes : ansible all -m copy -a "name=file1, dest=/tmp"
create user to all worker nodes : ansible all -m user -a "name =pavan state=present"
start the service in all worker nodes : ansible all - service -a"name=Jenkins state=started"
stope service in all worker nodes : ansible all - service -a"name=Jenkins state=stoped"

if you want to execute the multiple modules in all worker nodes: PLAYBOOKS

SETTING TOMCAT SERVER IN ANSIBLE PLYBOOK:

- hosts: all
  tasks:
    - name: installing tomcat
      get_url:
        url: "https://dlcdn.apache.org/tomcat/tomcat-9/v9.0.115/bin/apache-tomcat-9.0.115.tar.gz"
        dest: "/root/"

    - name: untar the tomcat file
      command: tar -xzvf apache-tomcat-9.0.115.tar.gz

    - name: installing java
      command: dnf install java-11-amazon-corretto -y

    - name: setting roles in tomcat
      template:
        src: tomcat-user.xml
        dest: /root/apache-tomcat-9.0.115/conf/tomcat-users.xml

    - name: deleting the lines
      template:
        src: context.xml
        dest: /root/apache-tomcat-9.0.115/webapps/manager/META-INF/context.xml

    - name: start the tomcat server
      shell: nohup /root/apache-tomcat-9.0.115/bin/startup.sh
~                                                               



DEPLOYING THE WAR FILE IN TOMCAT SERVER:
- hosts: all
  become: yes
  tasks:
    - name: copy war file to tomcat webapps
      copy:
        src: /root/sample.war
        dest: /root/apache-tomcat-9.0.115/webapps/sample.war
        mode: '0644'

    - name: restart tomcat
      shell: |
        /root/apache-tomcat-9.0.115/bin/shutdown.sh || true
        sleep 5
        /root/apache-tomcat-9.0.115/bin/startup.sh
